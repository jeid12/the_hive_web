# Font Installation

## Nature Beauty by Billy Argel Fonts

To complete the font setup, please download the **Nature Beauty** font:

1. Visit: https://www.dafont.com/nature-beauty.font
2. Download the font file
3. Extract the `.ttf` file
4. Rename it to `NatureBeauty.ttf`
5. Place it in this folder (`public/fonts/`)

### File Structure
```
public/
└── fonts/
    ├── NatureBeauty.ttf  ← Place the font file here
    └── README.md
```
